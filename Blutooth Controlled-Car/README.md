# Bluetooth-Controlled Car using ESP32

## Description
A wireless robotic car controlled using a mobile phone through Bluetooth communication between two ESP32 boards.

## Components Used
- ESP32 (Master & Slave)
- Motor Driver
- DC Motors
- Bluetooth (ESP built-in)
- Battery

## Working
The master ESP32 receives commands from a mobile app and sends them to the slave ESP32.  
The slave controls the motors using PWM signals for forward, reverse, left, right, and stop movements.

## Folder Structure
- Code – ESP32 programs
- Circuit-Diagram – Motor and ESP wiring
- Images – Car and setup photos
