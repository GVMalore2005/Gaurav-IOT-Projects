# ESP-to-ESP Emergency Traffic Signal System

## Description
A smart traffic control system where an ambulance unit communicates with traffic signals to give green light priority during emergencies.

## Components Used
- ESP32 Boards
- Traffic Signal LEDs
- Power Supply
- Wi-Fi Communication

## Working
When an ambulance is detected, the ESP unit in the ambulance sends a signal to the traffic signal ESP.  
The system automatically switches all lights to GREEN, allowing the ambulance to pass without delay.

## Folder Structure
- Code – ESP programs
- Circuit-Diagram – Signal wiring
- Images – Hardware setup and output
